# shared — 프론트·백엔드 공용

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../CLAUDE.md)의 **PRD** 섹션 참조.

디자인 문서의 **형식**과 **기하 규칙**을 양쪽이 공유하는 곳. MVVM의 **Model 레이어 토대**.

## 규칙
- UI·렌더링 코드(React, Canvas, R3F)를 절대 import하지 않는다.
- 순수 타입과 순수 함수만 둔다. 부수효과·네트워크 호출 금지.
- 프론트와 백엔드가 같은 디자인 문서 형식을 쓰도록 보장한다.

## 하위 폴더
- `schema/` — 디자인 문서 타입·검증
- `geometry/` — 전개도↔UV 매핑·좌표 변환
