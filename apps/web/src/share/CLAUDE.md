# share — 링크 진입·공유 (PRD-M5)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../CLAUDE.md)의 **PRD** 섹션 참조.

저장·링크 공유의 프론트 흐름. 비로그인 전제.

## 담는 것
- 편집 링크 진입: 작성자가 원본을 다시 열어 수정
- 열람 링크 진입: 비로그인 사용자가 3D 시안 열람, **복제 후 수정**
- 저장 트리거 후 발급된 두 URL(편집·열람) 노출

## 규칙
- 실제 토큰 발급·검증은 백엔드 `apps/api/src/share`. 여기는 진입·UI 흐름.
- 열람자는 원본을 못 바꾸고, 복제본을 새 디자인으로 만들어 편집.
