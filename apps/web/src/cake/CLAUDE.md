# cake — 시트 모양·색상·규격 (PRD-M1, PRD-M2, PRD-S5)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../CLAUDE.md)의 **PRD** 섹션 참조.

케이크의 형태 변수를 정하는 기능.

## 담는 것
- 시트 모양 선택: 원형/사각/하트 등 최소 3종 (PRD-M1)
- 색상 선택: 시트·크림 베이스 색, 팔레트 또는 컬러 피커 (PRD-M2)
- 규격 조정: 사이즈(호수)·높이·층수(단) (PRD-S5)

## 규칙
- 선택·조정 결과는 `document/store`에 반영 → 전개도·3D가 함께 갱신.
- 모양·규격 변경 시 전개도·UV 재계산은 `shared/geometry`가 담당.
