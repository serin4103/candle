# texture — 전개도→텍스처 굽기 (ViewModel, 동기화 핵심)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../../CLAUDE.md)의 **PRD** 섹션 참조.

candle의 전개도↔3D 동기화(PRD-M4)가 실제로 동작하는 곳.

## 담는 것
- `document/store` 구독 → 전개도를 오프스크린 캔버스에 렌더 → 텍스처 생성
- 생성한 텍스처를 `meshes/`의 메시에 UV 매핑으로 입힘
- 디자인 변경 시 재생성 (디바운스·부분 갱신 등 성능 고려)

## 규칙
- 여기만 견고하면 PRD-M4 동기화가 "공짜"로 동작한다 — 전환 시 별도 변환 로직 불필요.
- UV 매핑 좌표 규칙은 `shared/geometry`를 따른다.
