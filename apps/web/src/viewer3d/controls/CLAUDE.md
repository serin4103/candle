# controls — 카메라·3D 직접배치 (View + ViewModel)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../../CLAUDE.md)의 **PRD** 섹션 참조.

3D 뷰 조작과 3D 상에서의 요소 배치/조정.

## 담는 것
- 카메라: 360° 회전·확대축소 (PRD-M4)
- 3D 직접배치(PRD-S2): 옆면 경계·윗면처럼 전개도에서 잡기 어려운 위치 보정. **PRD-S3보다 우선**

## 규칙
- PRD-S2는 3D 포인터를 **전개도 좌표로 역변환**해 `document/store`에 반영해야 한다 — 이 역변환은 `shared/geometry` 사용. (까다로운 리스크 지점)
- 카메라 조작은 디자인 문서를 바꾸지 않는다 (표현 상태일 뿐).
