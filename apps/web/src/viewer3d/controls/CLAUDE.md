# controls — 카메라 (View + ViewModel)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../../CLAUDE.md)의 **PRD** 섹션 참조.

3D 뷰 카메라 조작. **현재 3D 뷰는 읽기 전용**(확인 전용)이다.

## 담는 것
- 카메라: 360° 회전·확대축소 (PRD-M4)

## 범위 밖 (Could로 이관)
- **3D 직접배치(PRD-S2 — Should→Could 이동)**: 3D 상에서 요소 배치/미세 조정(옆면 경계·윗면 보정). Could 단계에서 착수하며, 그때 3D 포인터를 **전개도 좌표로 역변환**(`shared/geometry`)해 `document/store`에 반영한다. (까다로운 리스크 지점 — PoC 우선.)

## 규칙
- 카메라 조작은 디자인 문서를 바꾸지 않는다 (표현 상태일 뿐).
