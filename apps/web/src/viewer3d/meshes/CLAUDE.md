# meshes — 케이크 메시 생성 (View)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../../CLAUDE.md)의 **PRD** 섹션 참조.

shape별 3D 케이크 형상.

## 담는 것
- 원형 = 실린더, 사각 = 박스, 하트 = 압출(extrude)
- 규격(호수·높이·층수, PRD-S5)에 따른 메시 치수
- 메시의 UV 좌표 (텍스처가 올바로 입혀지도록)

## 규칙
- UV 매핑 규칙 자체는 `shared/geometry`에 두고, 여기선 그 규칙으로 메시를 구성.
- 텍스처 생성은 `texture/`가 담당 — 여기는 형상만.
