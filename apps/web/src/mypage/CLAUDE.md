# mypage — 내 디자인 관리 (PRD-S6)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../CLAUDE.md)의 **PRD** 섹션 참조.

로그인 사용자가 저장한 디자인을 모아 보고 다시 열어 편집을 이어가는 프론트 화면.

## 담는 것
- 내 저장 디자인 목록 조회(`api`의 `listMyDesigns`) 및 표시(이름·수정일·썸네일 등)
- 목록에서 선택 시 해당 디자인 편집 페이지(`/d/:id`)로 이동

## 규칙
- 입력 위임·렌더만 하는 View. 목록 호출은 `api`, 인증은 `auth` 세션에 위임.
- 소유권 검증은 백엔드(`apps/api/src/auth`·`designs`)가 한다 — 여기서 접근 제어 판단 금지.
