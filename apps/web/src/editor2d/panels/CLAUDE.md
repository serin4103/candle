# panels — 속성·라이브러리 UI (View)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../../CLAUDE.md)의 **PRD** 섹션 참조.

요소 라이브러리와 속성 편집 패널.

## 담는 것
- 요소 라이브러리: 일러스트·레터링·파이핑 자산 목록, 검색·템플릿(PRD-C1)
- 속성 패널: 선택 요소의 폰트·색상·크기 등 편집 컨트롤

## 규칙 (View)
- 사용자 입력을 `document/store` 액션이나 `tools` 명령으로 전달만.
- 패널 자체에 편집 계산 로직을 두지 않는다.
