# canvas — 전개도 렌더링·입력 (View)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../../CLAUDE.md)의 **PRD** 섹션 참조.

전개도와 그 위 요소를 그리고, 선택 핸들을 표시하며, 포인터 이벤트를 받아 ViewModel(`tools`)에 위임한다.

## 담는 것
- 전개도·요소 렌더링, 선택 핸들(8개) 표시
- 히트 테스트: 어떤 요소/핸들을 눌렀는지 판별
- 포인터 이벤트를 `tools`의 명령으로 전달

## 규칙 (View)
- **계산 금지.** 스케일 수식·회전 보정·좌표 변환은 `tools`/`shared/geometry`로.
- 상태를 직접 들지 않고 `document/store`를 구독해 렌더만 한다.
