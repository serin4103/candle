# elements — 요소 정의·렌더링 (Model + View)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../../CLAUDE.md)의 **PRD** 섹션 참조.

전개도에 올라가는 요소 종류별 정의와 렌더링.

## 담는 것
- 카테고리: 일러스트, 레터링(텍스트 입력·폰트·색상), 파이핑 (PRD-M3)
- 손그림 획(PRD-S1), 업로드 이미지(PRD-S4)
- 각 요소의 기본 속성과 화면 표현

## 규칙
- 요소의 **데이터 형식**은 `shared/schema`를 따른다 (여기선 렌더링·UI 표현 위주).
- 이동/스케일 등 편집 동작은 여기 두지 말고 `tools`로 — 여기는 "무엇을 그리는가"만.
