# assets — 이미지 업로드·스토리지 (PRD-S4)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../CLAUDE.md)의 **PRD** 섹션 참조.

사용자가 올린 이미지 파일 처리.

## 담는 것
- 업로드 수신: PNG·JPG·SVG, 최대 50MB, 검수 없음
- 오브젝트 스토리지 저장, 메타(`Asset`) 기록
- 업로드된 이미지는 전개도에서 일반 요소처럼 사용

## 규칙
- 파일 크기·타입 검증은 경계에서 수행.
- 실제 저장은 `infra/` 오브젝트 스토리지 연동을 통해.
