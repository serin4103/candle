# share — 링크 토큰 발급·검증 (PRD-M5, PRD-C4)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../CLAUDE.md)의 **PRD** 섹션 참조.

비로그인 공유의 핵심. 인증 모듈을 대신해 접근 제어를 담당.

## 담는 것
- `editToken` 발급: 작성자가 원본 수정
- `viewToken` 발급: 비로그인 열람·복제
- 두 토큰은 **서로 다른 고유 URL**
- 토큰 검증, 만료·접근 제어 (PRD-C4)

## 규칙
- 편집 토큰과 열람 토큰의 권한을 명확히 분리 (열람 토큰으로 원본 수정 불가).
- 토큰은 추측 불가능하게 생성.
