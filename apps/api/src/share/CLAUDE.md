# share — 열람 링크 토큰 발급·검증 (PRD-M5, PRD-C4)

> 제품 요구사항(기능 ID 정의)은 루트 [CLAUDE.md](../../../../CLAUDE.md)의 **PRD** 섹션 참조.

비로그인 열람 공유의 핵심. 편집 권한은 소유권(PRD-S6, `auth`/`designs`)이 담당하고,
여기서는 열람(view) 토큰만 다룬다.

## 담는 것
- `viewToken` 발급: 비로그인 열람·복제
- 토큰 검증, 만료·접근 제어 (PRD-C4)
- (편집 링크 `editToken`은 PRD-S6에서 제거 — 편집은 소유권으로 제어)

## 규칙
- 열람 토큰으로는 원본을 수정할 수 없다(편집은 소유자만).
- 토큰은 추측 불가능하게 생성.
